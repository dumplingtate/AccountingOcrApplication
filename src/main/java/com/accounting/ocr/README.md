# Document OCR System

Система автоматического распознавания и классификации бухгалтерских документов с использованием компьютерного зрения.

## 🚀 Возможности

- Загрузка документов (изображения, PDF)
- Автоматическое распознавание текста (OCR)
- Классификация типов документов (счета, акты, накладные)
- Извлечение ключевых реквизитов (номер, дата, сумма, ИНН)
- Валидация извлеченных данных
- Хранение истории документов
- REST API для интеграции

## 🛠 Технологический стек

### Backend (Java)
- Spring Boot 3.x
- Spring Data JPA
- PostgreSQL / H2
- REST API

### ML Service (Python)
- FastAPI
- Tesseract OCR
- OpenCV
- scikit-learn

### DevOps
- Docker
- Docker Compose

## 📋 Требования

- Docker 20.10+
- Docker Compose 2.0+
- 4GB RAM
